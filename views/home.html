About UNCA Course Finder

This application allows you to create a course schedule for the next semester.

To add classes to the "My Schedule" section simply click on the desired class from the large table below. To remove classes, click on the one you want to remove in the "My Schedule" section. For more info on how to use this application click on the link above.

This application may not work properly on mobile devices. Use a computer for best results.
